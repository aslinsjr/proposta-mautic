1. Objetivo

Implementação de infraestrutura dedicada para automação de marketing utilizando o Mautic 5, garantindo independência de plataformas SaaS, soberania dos dados e redução expressiva de custos operacionais.

2. Escopo da Implementação

  • Infraestrutura: Setup de Servidor VPS (Linux), Nginx, PHP 8.1 e SSL.
  • Mautic Core: Instalação, banco de dados e otimização de Cron Jobs.
  • Entregabilidade: Amazon SES + DNS (SPF, DKIM, DMARC).
  • Tracking: Script de rastreamento e pixel de monitoramento.
